# xkApp

# How to create demo app with xkutils

```bash
mkdir xkApp
cd xkApp
git clone git@github.com:liuxk99/xkApp.git ./
mkdir xkutils
cd xkutils
git clone git@github.com:liuxk99/xkutils.git ./
```
